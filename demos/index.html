Janus Demos

Portable, reproducible demonstrations of Janus Core Lite invariants.

  • Shared contract — input/output conventions and result schema.
  • Node demo runner — minimal scaffold for replay + reporting.
  • Python telemetry demo — minimal scaffold for ingesting signals and deriving evidence.
  • CSV portable demo — fixture-first verification using CSV and tiny scripts.
  • Stress reproducibility demo — scaffold for generating load and verifying rebuild consistency.

Note: This repo is scaffold-only initially. The runners are placeholders until the full demo logic is implemented.
